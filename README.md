#租房后台管理系统介绍
系统主要模块有房源管理，租房管理，用户管理，三大块。
架构：SpringBoot+Springmvc+Dubbo+Mybatis
中间件使用:
mysql,redis,mongodb,zookeeper,Elasticsearch,
rocketmq,nginx。
